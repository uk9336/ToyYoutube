## ToyYoutube
- - -
* 개발환경
  - IDE : Android Studio
  - 사용언어 : Kotlin
  - 사용기술 : Binding, Retrofit, Glide, ExoPlayer
  - UI : RecyclerView, MotionLayout, PlayerView
- - -
